# Task Queue System — Future Module

## Purpose
Structured task management for clinical follow-ups, orders, and coordination items.

## Task Types
- Lab orders
- Imaging requests
- Specialist consultations
- Medication changes
- Patient transport
- Discharge planning items
- Follow-up appointments

## Key Features
- Tasks linked to patients and medical records
- Priority levels with visual indicators
- Assignee tracking
- Status workflow (Pending → In Progress → Completed)
- Due date tracking
- Source tracking (manual vs AI-extracted)
- Dashboard widget for pending tasks

## Integration
- AI agent can create tasks from note analysis
- Activity logger tracks task lifecycle
- Permission system controls who can create/assign/complete tasks
